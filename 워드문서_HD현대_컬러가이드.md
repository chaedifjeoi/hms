# HD현대마린솔루션테크 워드 문서 컬러 & 스타일 가이드

> `generate_word.py`에서 추출한 **공식 CI 컬러 + 실제 적용 패턴**을 정리한 문서.
> 워드 문서를 만들 때 이 팔레트와 스타일 규칙을 따르면 회사 아이덴티티와 통일된 문서가 나옴.

## 2. 컬러 적용 규칙 (generate_word.py 기반)

### 2-1. 제목 계층별 컬러

| 계층 | 스타일 | 컬러 | 크기 | 기타 |
|------|--------|------|------|------|
| **섹션 제목** | `add_section_title()` | 번호: 흰색 on #008233 배경 / 제목: #008233 | 14pt(번호) + 16pt(제목) | 번호에 초록 배경 하이라이트 |
| **서브 제목** | `add_sub_title()` | #008233 (Prosperity Green) | 13pt | `▎` 접두사 |
| **서브서브 제목** | `add_sub_sub_title()` | #00AD1D (Heritage Green) | 11pt | `●` 접두사 |

| **불릿** | `add_bullet()` | `▸` 문자: #00AD1D / 텍스트: #2C2C2C | 10pt | - |

## 6. matplotlib 그래프 공통 스타일

```python
import matplotlib.pyplot as plt

plt.rcParams['font.family'] = 'AppleGothic'
plt.rcParams['axes.unicode_minus'] = False

# 공통 그래프 스타일 함수
def apply_hd_chart_style(ax, title):
    """HD현대 스타일을 matplotlib 차트에 적용"""
    ax.spines['top'].set_visible(False)
    ax.spines['right'].set_visible(False)
    ax.grid(axis='y', alpha=0.3, linestyle='--')
    ax.set_axisbelow(True)
    ax.set_title(title, fontsize=14, fontweight='bold', pad=15, color='#008233')
```

## 7. 기존 가이드 → HD현대 컬러 매핑

> 기존 `워드문서_가독성_가이드.md`와 `워드문서_python-docx_레시피.md`는 네이비(#1B3A5C) 기반.
> 아래 매핑 테이블로 바꿔서 사용하면 HD현대 스타일이 됨.

| 기존 가이드 컬러 | 용도 | → HD현대 컬러 | Hex |
|-----------------|------|--------------|-----|
| #1B3A5C (네이비) | 제목, 표 헤더, 메인 강조 | **HD Prosperity Green** | `#008233` |
| #2B579A (블루) | 소제목, 보조 강조 | **HD Heritage Green** | `#00AD1D` |
| #E74C3C (레드) | 위험/경고 | (동일) | `#E74C3C` |
| #27AE60 (그린) | 긍정/완료 | **HD Prosperity Green** | `#008233` |
| #F0F4F8 (연한 블루) | 표 교대행, 박스 배경 | **Light Green** | `#E8F5E9` |
| #333333 (다크 그레이) | 본문 텍스트 | **Dark Gray** | `#2C2C2C` |
| #E8F0FE (연한 블루 배경) | 인사이트 박스 | **Light Green** | `#E8F5E9` |

### 변환 예시 — 레시피 코드에서 컬러만 교체

```python
# 기존 (네이비 기반)
NAVY    = '1B3A5C'
BLUE    = '2B579A'
LIGHT   = 'F0F4F8'

# HD현대 (초록 기반) — 이렇게만 바꾸면 됨
NAVY    = '008233'   # → Prosperity Green
BLUE    = '00AD1D'   # → Heritage Green
LIGHT   = 'E8F5E9'   # → Light Green
```

## 9. 컬러 사용 비중 가이드

문서 전체에서 각 컬러가 차지하는 시각적 비중:

| 컬러 | 비중 | 역할 |
|------|------|------|
| **#008233 (Prosperity Green)** | 15~20% | 제목, 표 헤더, 인사이트 박스, 그래프 메인 |
| **#00AD1D (Heritage Green)** | 5~8% | 서브 강조, 불릿, 그래프 보조 |
| **#E8F5E9 (Light Green)** | 10~15% | 박스 배경, 하이라이트 배경 |
| **#2C2C2C (Dark Gray)** | 50~60% | 본문 텍스트 (가장 많은 면적) |
| **#F5F5F5 / #F8F9FA** | 8~10% | 표 교대행 배경 |
| **#E74C3C / #F39C12** | 2~3% | 상태 시그널 (절제해서 사용) |
| **#FFFFFF (White)** | 나머지 | 페이지 배경, 표 헤더 글씨 |

> **핵심**: 초록이 "포인트"로 돋보이려면, 본문 대부분은 다크 그레이/화이트로 차분하게 유지해야 함.
> 초록을 과도하게 사용하면 오히려 강조 효과가 사라짐.
