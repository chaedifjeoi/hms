# 라이프사이클 계약(Lifecycle Agreement) 도입 제안

## HD현대마린솔루션테크 배전반 AM 사업의 수익 구조 전환

---

# 서론 — 왜 필요한가

## 1. 현재 수익 구조의 문제

우리 회사의 현재 배전반 AS 비즈니스 모델은 **"건별 일회성 수익"** 구조입니다.

```
선주 연락 → 점검/수리 → 비용 청구 → 끝 → (다음 연락이 올 때까지 대기)
```

이 구조의 근본적 문제:

| 문제 | 상세 |
|------|------|
| **매출 불규칙** | 고장이 나야 수익. 언제 얼마가 들어올지 예측 불가 |
| **수동적 영업** | 선주가 연락해야 AS 시작. 우리가 주도권을 가질 수 없음 |
| **고객 이탈 위험** | 건별 거래이기 때문에, 다음 건에서 경쟁사로 갈 수 있음 |
| **가격 경쟁** | 건별 견적이라 매번 가격 비교 대상이 됨 |
| **관계 단절** | AS 건이 없는 기간에는 고객과의 접점이 없음 |

## 2. 선도기업은 이미 전환했다

**바르질라(Wärtsilä)**는 AM 사업의 핵심 전략으로 **Lifecycle Agreement(라이프사이클 계약)**를 적극 추진하고 있습니다.

### 바르질라 라이프사이클 계약에 포함되는 것

| 항목 | 내용 |
|------|------|
| **정비 계획 수립** | Dynamic Maintenance Planning — 실제 데이터 기반으로 정비 주기 최적화 (유지보수 비용 14% 절감 실적) |
| **24/7 원격 운영 지원** | 전문가가 원격으로 상시 모니터링 + 문제 발생 시 즉시 대응 |
| **Expert Insight** | AI 기반 예지정비 — 실시간 장비 모니터링 → 문제 발생 전 사전 감지 |
| **정비 부품** | 예정된 정비에 필요한 부품 포함 (부품 수배 고객 부담 없음) |
| **현장 서비스** | 대규모/복잡 작업 시 현장 엔지니어 파견 |
| **교육** | 선원 교육 서비스 |
| **전담 매니저** | 계약 고객에게 전담 Contract Manager 배정 (대형 계약) |
| **성능 보증** | 장비 성능 목표치 보증 |

### 바르질라가 이렇게 하는 이유

> "건별 수리로 돈 버는 것보다, 장기 계약으로 **안정적 반복 매출**을 만드는 게 더 가치 있다"

실제로 바르질라는 서비스 사업에서 **"서비스 가치 사다리(Service Value Ladder)를 올라간다"**는 표현을 사용합니다:

```
[바르질라 서비스 가치 사다리]

Level 1: 부품 판매 (Parts)              ← 가장 단순, 마진 낮음
Level 2: 현장 서비스 (Field Service)     ← 건별 수리
Level 3: 유지보수 계약 (Maintenance)     ← 정기 계약
Level 4: 라이프사이클 계약 (Lifecycle)   ← 통합 관리, 마진 높음 ★
Level 5: 성과 보증 (Performance)         ← 최상위, 성과 기반 과금
```

**우리 회사는 현재 Level 2(건별 수리)에 머물러 있습니다. 바르질라는 Level 4~5로 올라가고 있습니다.**

# 본론 — 도입 방안과 기대 이득

## 1. 우리 회사에 맞는 라이프사이클 계약 설계

바르질라 모델을 그대로 복사할 수는 없습니다. 우리는 배전반/전기 기자재 전문이고, 자재는 HMS를 통해 공급합니다. 우리 상황에 맞게 설계해야 합니다.

### 계약 구조 — 3개 티어

#### Tier 1. Basic — "정기 점검 계약"

> 진입 장벽 낮음. 기존 무상 헬스체크를 유상화하는 첫 단계.

| 포함 항목 | 내용 |
|----------|------|
| **연간 정기 헬스체크** | 연 1~2회 엔지니어 방문 점검 (입거 시) |
| **시각화 리포트** | 그래프/대시보드 포함 고품질 점검 리포트 제공 |
| **기술 문서 접근** | 웹 포털에서 해당 선박 매뉴얼, 도면 24/7 열람 |
| **점검 알림** | 점검 시기 도래 시 자동 리마인더 |
| **웹 포털 접근** | 호선정보, 이력 조회, 서비스 요청 기능 |

| 항목 | 내용 |
|------|------|
| **가격 구조** | 선박당 연간 고정 요금 |
| **계약 기간** | 1~2년 |
| **대상 고객** | 전체 고객 (진입 단계) |

#### Tier 2. Standard — "종합 관리 계약"

> 핵심 티어. 정기 점검 + 기술 지원 + 우선 대응까지 포함.

| 포함 항목 | 내용 |
|----------|------|
| **Tier 1 전체** | 정기 헬스체크 + 리포트 + 문서 + 알림 + 웹 포털 |
| **기술 지원** | 전화/웹/이메일 기술 문의 무제한 (응답 시간 보증: 24시간 이내) |
| **우선 대응** | 긴급 수리 요청 시 비계약 고객 대비 우선 배정 |
| **부품 수명 관리** | 부품별 잔여 수명 추적 + 교체 시기 선제 알림 |
| **리포트 고도화** | AI 분석 코멘트 + 추이 분석 + 향후 전망 포함 리포트 |
| **전담 PM** | 해당 고객 전담 PM 배정 (커뮤니케이션 일원화) |

| 항목 | 내용 |
|------|------|
| **가격 구조** | 선박당 연간 고정 요금 (Tier 1 대비 2~3배) |
| **계약 기간** | 2~3년 |
| **대상 고객** | 다수 선박 보유 선주, 관리사 |

#### Tier 3. Premium — "토탈 케어 계약"

> 최상위. 수리 서비스까지 포함한 올인원 패키지.

| 포함 항목 | 내용 |
|----------|------|
| **Tier 2 전체** | 종합 관리 + 기술 지원 + 우선 대응 + 리포트 |
| **수리 서비스 포함** | 계약 범위 내 수리 작업 비용이 연간 요금에 포함 (정액제) |
| **긴급 대응 보증** | 긴급 수리 요청 시 N시간 이내 대응 보증 (SLA) |
| **원격 기술 지원** | 화상/AR 원격 진단 지원 |
| **정비 계획 수립** | 연간 정비 계획을 우리가 수립 + 관리 (고객은 승인만) |
| **데이터 분석 리포트** | 분기별 선대 전체 배전반 건강도 분석 리포트 |
| **교육** | 선상 기관사 대상 배전반 기본 점검 교육 (연 1회) |

| 항목 | 내용 |
|------|------|
| **가격 구조** | 선박당 연간 고정 요금 (가장 높지만, 고객은 유지보수 비용 예측 가능) |
| **계약 기간** | 3~5년 |
| **대상 고객** | 대형 선주, 장기 관계 고객 |

## 2. 도입 시 우리 회사에 돌아오는 이득

### 이득 1. 매출 구조 전환 — 불규칙 → 안정적 반복 매출

| | 현재 (건별 수익) | 라이프사이클 계약 후 |
|---|---|---|
| **매출 예측** | 불가 (고장 나야 수익) | **연간 계약 금액 확정** |
| **매출 패턴** | 들쭉날쭉 | **월/분기별 안정적** |
| **영업 방식** | 건마다 새로 영업 | **계약 갱신 중심** (갱신율이 핵심 KPI) |

**예시 시뮬레이션:**
```
Tier 2 Standard 기준
• 선박당 연간 계약금: ₩20M (가정)
• 계약 선박 50척
• 연간 반복 매출: ₩20M × 50척 = ₩10억

이건 고장이 나든 안 나든 들어오는 매출입니다.
여기에 Tier 3 수리 포함 계약이 추가되면 규모가 더 커집니다.
```

### 이득 2. 고객 이탈 방지 (Lock-in 효과)

| | 현재 | 라이프사이클 계약 후 |
|---|---|---|
| **전환 비용** | 없음 (건별이라 다음 건에 경쟁사 가능) | **계약 기간 내 이탈 없음** |
| **관계 깊이** | 건별 거래 (얕음) | **전담 PM + 정기 방문 + 데이터 축적** (깊음) |
| **데이터 축적** | 건별로 흩어짐 | **연속 데이터 → 우리만 가진 이력** (경쟁사 진입 장벽) |

**핵심:** 우리가 3년간 쌓은 배전반 데이터를 경쟁사는 갖고 있지 않음 → 선주가 우리를 바꿀 이유가 줄어듦

### 이득 3. 서비스 가치 사다리 상승 → 마진 개선

| | 건별 수리 | 라이프사이클 계약 |
|---|---|---|
| **가격 경쟁** | 매번 견적 비교 대상 | **가치 기반 가격** (편의 + 안정 + 품질 포함) |
| **마진** | 경쟁으로 마진 압박 | **패키지 프리미엄** 가능 |
| **업셀** | 수리 건만 | 헬스체크 → Standard → Premium **업그레이드 유도** |

### 이득 4. 선제적 영업 → 추가 수주

| | 현재 | 라이프사이클 계약 후 |
|---|---|---|
| **데이터** | 건별 데이터 (단편적) | **연속 데이터 축적** |
| **영업 방식** | 고객이 연락해야 시작 | 데이터 기반 **"이 부품 곧 교체 시기입니다" 선제 제안** |
| **추가 매출** | 없음 | 계약 범위 외 작업 발생 시 **자연스럽게 우리에게 발주** |

### 이득 5. 무상 헬스체크 비용 회수

| | 현재 | 라이프사이클 계약 후 |
|---|---|---|
| **헬스체크 비용** | 무상 (우리 부담) | **계약 요금에 포함** (유상화) |
| **리포트** | 무상으로 써줌 | 고품질 리포트가 **계약의 가치** |

**핵심:** 지금 무상으로 하고 있는 헬스체크를 "무상이라 가치를 못 느끼는 것"에서 "계약에 포함된 프리미엄 서비스"로 포지셔닝 전환

## 3. 구체적 도입 프로세스

### Step 1. 파일럿 고객 선정 (1~2개월)

| 활동 | 상세 |
|------|------|
| **대상 선정** | 기존 관계가 좋은 선주 2~3곳 선정 (선박 5~10척 규모) |
| **선정 기준** | ① 다수 선박 보유 ② 정기적으로 AS 요청하는 고객 ③ 장기 관계 |
| **사전 조사** | 해당 고객 선박의 과거 AS 건수, 비용, 빈도 분석 → 계약 가격 산정 근거 |
| **내부 준비** | 계약서 템플릿 작성, 서비스 범위 확정, 가격 시뮬레이션 |

### Step 2. Tier 1 (Basic) 계약 제안 (2~3개월)

| 활동 | 상세 |
|------|------|
| **제안서 작성** | "연간 정기 점검 + 시각화 리포트 + 웹 포털 접근" 패키지 |
| **가격 제안** | 과거 AS 비용 분석 → "이 가격이면 건별 대비 절감됩니다" 논리 |
| **고객 미팅** | "바르질라, 콩스버그는 이미 이렇게 하고 있습니다" 벤치마킹 자료 활용 |
| **계약 체결** | 1~2년 시범 계약 |

**고객 설득 포인트:**

| 고객이 얻는 것 | 상세 |
|--------------|------|
| **비용 예측 가능** | "연간 ₩OOM이면 배전반 유지보수 걱정 끝" |
| **프리미엄 리포트** | 그래프/AI 분석 포함 고품질 리포트 (건별로는 안 줌) |
| **웹 포털** | 호선정보, 이력, 문서를 24/7 직접 조회 |
| **우선 접근** | 비계약 고객보다 우선 대응 |
| **안심** | "관리 받고 있다"는 심리적 안정감 |

### Step 3. 실행 + 데이터 축적 (6~12개월)

| 활동 | 상세 |
|------|------|
| **헬스체크 수행** | 계약에 따라 정기 점검 실시 |
| **리포트 발행** | 시각화 리포트 웹 포털 업로드 |
| **데이터 축적** | 측정값, 이력, 부품 상태 데이터 체계적 기록 |
| **고객 피드백** | 분기별 고객 만족도 확인 + 개선사항 반영 |
| **업셀 기회 탐색** | "이 부품 교체 시기 접근 중" → 추가 수리 건 발생 → 자연스러운 수주 |

### Step 4. 확대 + 업그레이드 (12개월~)

| 활동 | 상세 |
|------|------|
| **파일럿 성과 정리** | 고객 만족도, 수주 증가율, 반복 매출 규모 등 성과 수치화 |
| **성공 사례화** | "이 선주는 라이프사이클 계약 후 유지보수 비용 N% 절감" → 영업 자료 |
| **대상 확대** | 파일럿 성과 기반으로 다른 고객에게 제안 확대 |
| **Tier 업그레이드** | Tier 1 고객 → Tier 2 Standard로 업그레이드 제안 |
| **장기 비전** | Tier 3 Premium까지 라인업 완성 |


# 결론 — 왜 지금 해야 하는가

## 1. 시장이 움직이고 있다

바르질라는 2024~2026년에만 **수십 척 규모의 라이프사이클 계약**을 연이어 체결하고 있습니다. 아일랜드 해군 4척, Seapeak LNG 10척, CMA Ships 14척, 중동 준설사 5년 계약, LNG 운반선 14척, 12척...

**이건 트렌드가 아니라, 이미 표준이 되고 있습니다.**

## 2. 우리가 유리한 위치에 있다

- **HD현대 건조 선박의 유일한 AS 사업자** — 독점적 지위
- **9,890척 네트워크** — 계약 제안할 고객 풀이 이미 있음
- **헬스체크 모델** — 이미 정기 방문하고 있음 → 계약화하기 가장 쉬운 구조
- **기술력** — 배전반 전문 엔지니어링 역량 보유

## 3. 안 하면 벌어지는 일

- 선도기업과의 서비스 모델 격차가 **더 벌어짐**
- 경쟁 AS 업체가 라이프사이클 계약을 먼저 제안하면 **고객을 빼앗김**
- 건별 수익 구조에서 벗어나지 못하면 **매출 예측 불가 + 마진 압박** 지속

## 4. 한 문장 요약

> **"지금 무상으로 하고 있는 헬스체크를, 라이프사이클 계약이라는 그릇에 담으면,**
> **같은 일을 하면서 안정적 반복 매출을 만들 수 있다.**
> **바르질라는 이미 하고 있고, 우리는 할 수 있는 위치에 있다.**
> **문제는 '할 수 있느냐'가 아니라, '언제 시작하느냐'다."**
